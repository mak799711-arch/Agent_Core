## Фаза 3 — PWA для кассира (Неделя 4–5)

Цель: то, что ты придумал сегодня — одна кнопка на планшете кассира.

- [ ] **3.1** Создать страницу `/cashier` — минимальный интерфейс:
  - Одно поле: ввод 6-значного кода
  - Одна кнопка: «Проверить»
  - Результат: имя агента + сумма скидки + кнопка «Подтвердить»
- [ ] **3.2** Добавить PWA-манифест (`manifest.json`) — чтобы можно было «Добавить на главный экран»
- [ ] **3.3** Сделать страницу работающей офлайн (service worker для базового кэша)
- [ ] **3.4** Протестировать на реальном планшете/телефоне — открыть, добавить на экран, проверить

> [!TIP]
> Это твоя «убийственная фича». Владелец кафе видит одну иконку на планшете → тыкает → вводит код → готово. Никаких сложных панелей.
